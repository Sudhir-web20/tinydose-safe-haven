# TinyDose Vault

A calm, careful medicine expiry tracker for your baby's cabinet.

TinyDose Vault helps parents and caregivers keep a quiet watch over every little
bottle, syrup, and sachet — so you always know what's safe to give and what
needs to be safely discarded.

## What it does

- 📦 **Track every medicine** — name, type, batch and expiry at a glance
- 🟢 **Smart status** — Safe, Expiring soon, Critical, or Expired
- 🗓️ **Calendar view** — see upcoming expiries by month
- 🔎 **Autocomplete** — quick entry from a built-in pediatric medicine list
- 🖼️ **Real product photos** — fetched from the web as you type the name
- ✅ **Mark as finished** — clean vault, kept history
- 💾 **Local-first** — your data stays in your browser

## Tech

Built with TanStack Start (React 19 + Vite 7), Tailwind v4, Framer Motion,
Zustand, and shadcn/ui.

## Develop

```bash
bun install
```
